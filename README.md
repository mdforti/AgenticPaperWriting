# Paper Writing Lab — MLFeMoTCPs Scientist Agents

Agent system for collaborative scientific paper writing based on the [MLFeMoTCPs](https://github.com/AIIMProject/MLFeMoTCPs) repository.

## Agents

| Role | File | Description |
|---|---|---|
| **author1** | `agents/author1.md` | Lead author, ML practitioner, notebook creator |
| **author2** | `agents/author2.md` | Senior author, advisor, atomistic simulations expert |
| **orchestrator** | `agents/orchestrator.md` | Workflow coordinator and paper submission manager |
| **reviewer1** | `reviewers/reviewer1.md` | Computational materials physicist — DFT, tight-binding, BOP descriptors |
| **reviewer2** | `reviewers/reviewer2.md` | ML for materials science — descriptor engineering, model validation |
| **reviewer3** | `reviewers/reviewer3.md` | Thermodynamics & CALPHAD — TCP phase stability, CEF |

## Skills

| Skill | File | Description |
|---|---|---|
| **PRB Author Guide** | `skills/prb_author_guide.md` | Complete formatting & submission guidelines for Physical Review B |

## Workflow

1. Analyze notebooks → extract findings
2. Outline paper sections (assign to authors)
3. Draft in parallel (Methods/Results → author1; Intro/Discussion/Conclusions → author2)
4. Integrate, review, iterate
5. Submit to journal (PRB — see `skills/prb_author_guide.md`)
6. Peer review by 3 independent reviewers → `reviewers/` feedback
7. Revise & resubmit based on reviewer comments
